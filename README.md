# 地铁计算器  
用于在上海地铁从车体号计算车号。

## 编译  
非常正常的安卓应用项目。  
## 功能  
- 从车体号计算车号  
- 查看计算公式  
- 自动检查更新(但需要手动下载)  
- 夜间模式  
  
# 地铁相关  
## 什么是车体号?  
指列车外部侧面(一般是车窗下方)的数字。部分列车车内也有。  
## 为什么要计算?
有时你只看到了列车侧面(如果你站在站台上观察多半如此)，想要知道车号，那么你需要本应用。  
本应用并非不可替代。你也可以手算。  
  
***本应用不代表上海地铁官方立场!***
